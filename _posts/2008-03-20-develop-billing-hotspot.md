> Tulisan ini di-import dari [blog WordPress saya yang lama](https://firstly.wordpress.com/2008/03/20/develop-billing-hotspot/), ditulis pada 2008-03-20.

Hari ini saya di minta sama Arman satari untuk membuat Billing HotSpot yang nantinya digunakan di warnetnya.

Sekedar Info aja bahwa perminta si Arman Satari “[Ketua LUGU](http://makassar.linux.or.id "Linux Makassar")” Segede ini :

  1. Tidak direpotkan lagi dengan harus membeli Server khusus
  2. Tidak direpotkan lagi dengan harus Instalasi & konfigurasi PC server ataupun peralatan Wireless access point
  3. Tidak perlu lagi membeli peralatan khusus Wireless AP
  4. Tidak direpotkan lagi dengan urusan kabel
  5. Tidak memerlukan monitor lagi, karena web based manajemen via browser dan dilakukan secara remote dimana saja
  6. Instant karena Plug & Play & Broadcast
  7. Harga sudah termasuk Hardware, Bandingkan harga kami dengan software Billing Hotspot lainnya yang tidak termasuk hardware
  8. Bisa dijadikan personal webserver (misalnya membuat gallery web foto-foto perusahaan) dan streming media server sehingga client bisa nonton atau dengar music online dari server billing cukup buka alamat url lalu ada sejumlah play list
  9. Billing di bangun di atas system operasi Linux dengan based system chillispot,
  10. php&mysql, PHP-Myprepaid, dan Freeradius
  11. Instan PC Hotspot (harga sudah termasuk Hardware)
  12. Bekerja di jaringan LAN dan Wireless/Wifi ( 1 AP, 2 Lan Adapter)
  13. User akan diarahkan langsung Redirect ke halaman Web-based login sebelum mengakses internet
  14. Membuat prepaid user accounts dengan username and password
  15. Mengontrol waktu pemakaian
  16. Pembatasan Bandwidth (Traffic Quota Management)
  17. Pengaturan kuota Bandwidth (Traffic Shaping Management)
  18. Anti Mac Cloning dan Anti hacking atau Wireless Hacking. Sudah teruji….
  19. Support DHCP dan static routing –
  20. Halaman Login bisa diedit Sesuai Logo dan keinginan Client
  21. Easy, powerful, reliable & user friendly bahkan yang awam sekalipun.
  22. URL Filtering
  23. Packet Filter
  24. Pengaturan Free open Web Site
  25. Dapat digunakan diatas Internet Explorer, Firefox, Opera, Safari, Konqueror dan lain-lain
  26. Tidak tergantung pada sistem hardware tertentu
  27. Billing berdasarkan pengaturan bandwidth (cocok untuk Warnet / RT/RW-Net)
  28. Harga kami termasuk yang termurah
  29. Log pemakaian setiap user
  30. Otomatis disconect jika waktu/bandwidth sudah habis
  31. Otomatis disconect bila user menutup browser
  32. Menampilkan report pemakaian setiap user selesai
  33. Prosentase pemakaian user kelihatan di client
  34. Account user bisa dibuat single user maupun multiuser
  35. Status pemakaian user muncul di Hotspot Manager
  36. Sisa pemakaain time/volume muncul di client
  37. Total dan batas pemakain user muncul di client
  38. Laporan pemakaian berdasarkan account user
  39. Laporan pemakaian berdasarkan waktu (time base)
  40. Laporan pemakaian berdasarkan bandwidth (volume base)
  41. Report/laporan penjualan voucher harian, mingguan, bulanan
  42. Account Generator (membuat account dengan cepat)
  43. Voucher Generator (membuat voucher secara massal)
  44. Print bisa secara remote, dimana saja.
  45. Login server & client berbasis web
  46. Support private IP lewat NAT dan IP public
  47. Kompatibel dengan standard IEEE 802.11xx
  48. Kompatibel dengan jenis hardware WLAN
  49. Pengaturan free open website
  50. Redirect ke website tertentu setelah user login
  51. Pengaturan idle time pada auto logout user
  52. Pengaturan bandwidth masing-masing account user
  53. Support DHCP dan static routing
  54. Menentukan area bebas akses (free surfing area)
  55. Web-based secara terpusat, bisa di remote dari manapun
  56. Authentikasi menggunakan standard RADIUS
  57. Support MAC/IP address block
  58. Bisa di Costumize Lebih lanjut lagi
